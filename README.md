# Facets (facets)

Facets is a platform engineering and infrastructure automation platform that unifies provisioning, deployment, and configuration into a single declarative model for governed, secure, and self-serve cloud operations. The Facets Control Plane exposes a REST API for managing blueprints, environments, releases, resources, and integrations across the platform.
